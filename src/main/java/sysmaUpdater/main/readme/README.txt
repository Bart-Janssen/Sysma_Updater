How to update:

# Sysma.xml en SysmaUpdater.xml zijn de de bestanden voor Launch4j. Deze staat geinstalleerd op de VM testVM.
1 Plak het .jar bestand op desktop en open het programma Launch4j.
2 Open in het programma Launch4j het bijhorende .xml bestand.
3 Klik op het tandwieletje.
# De .exe moet nu op het bureadblad tevoorschijn komen, dit is nu de bruikbare .exe.
